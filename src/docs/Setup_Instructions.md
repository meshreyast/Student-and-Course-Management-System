JDK Version used is: 25.0.2

How to run "Hello World" program: 

In the src directory go to Main.java file and 
write "Hello World!" in the main method of 
Main class using the System.out.println statement. 

Run this Main class in terminal using the IDE's 
dedicated option or run these commands
"javac Main.java". This command will create a 
file Main.class which is the Main.java file but in bytecodes.
Then run "java Main" command. The output will be printed 
in the terminal.

If you don't want to make a new bytecode file,
use the run button in your IDE.
