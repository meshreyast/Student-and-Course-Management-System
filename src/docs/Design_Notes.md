Why ArrayList instead of arrays?

Arrays have a fixed size and the size needs to be declared upfront.
In this console app, we don't know how much data the user will input. 
ArrayList grows and shrinks automatically making them ideal for this use case.
ArrayList also provide powerful methods out of the box without writing extra logic and ork seamlessly with enhanced for-loops and streams.

Static variables were used to keep a single copy each of the counter variables and their getter functions in the project.
Static was also used for util functions.
